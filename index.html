The basic Language of the web: HTML

Posted by Laura Jones on monday, june21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML,CSS and JavaScript. These are the language of the web

in this post, let's focus on HTML we will learn what html is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperTextMarkupLanguage. It's a markup language that web developers use to structure and describe the contentof a webpage (not a programming language)

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, videos, etc... web browsers understand HTML and render HTML code as website
